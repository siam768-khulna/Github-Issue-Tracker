1. What is the difference between var, let and const?
    Answer: var, let and const are used to declare variables in JavaScript.
var:It is the old way to declare variables. It can be re-declared and re-assigned.
let: It cannot be re-declared, but the value can be re-assigned.
const: It cannot be re-declared or re-assigned. The value remains constant after declaration.

2. What is the Spread Operator (...)?
   The spread operator (...) is used to expand or copy elements of an array or object
